# KACATKA-PROPHECY
гороскоп микросервис

### Запуск
```
> node server.js
```

### Открыть в браузере
```
http://localhost:3000
```

### Цель
* Научиться модулям
* Научиться npm
* Познакомится со встроенными javascript объектами Math и Date
* Подтянуть массивы
* Познакомиться с модулями http и fs (передача и хранение данных)
